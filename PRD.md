## 产品定位

面向熟悉或学习 **六爻纳甲** 的用户，提供 **在线摇卦、自动排盘、查看本卦/变卦、神煞与动静** 等的 Web 体验。  
**前端**：交互与展示；**后端（Spring Boot）**：装卦入库、排盘数据与业务规则。

---

## 核心功能

| 功能 | 说明 | 备注（随版本更新） |
|------|------|---------------------|
| 起卦 | 模拟铜钱摇卦得到六位爻象，提交后端生成记录 | 已有 |
| 排盘结果 | 四柱、旬空、本卦/变卦、六兽、六亲、动爻等展示 | 已有 |
| 账号与历史 | 注册/登录（JWT）、按用户查看摇卦历史 | 与后端 Security 对齐，见 architecture |
| 用神分析（规划） | 结果页选爻、调用后端用神接口 | 见 architecture 接口表 |

---

## 用户流程

1. **起卦**：进入首页 → 可选填写所问（标题）→ 完成摇卦 → 提交排盘 → 跳转结果页（携带记录 id）。
2. **结果**：查看四柱与卦象详情；返回首页或进入历史（接口就绪后）。
3. **历史**（能力就绪后）：列表展示过往记录 → 点击条目进入对应结果页。
4. **需登录后起卦时**（若产品启用）：先完成注册/登录（`identifier` + 密码，与后端一致）→ 再执行起卦；请求携带 `Authorization: Bearer`（实现细节见 **architecture.md**）。
